# Code-card-generator
This is a code card generator that can automatically generate code cards with syntax highlighting based on the source code files you upload. There are two themes to choose from: bright and dark, with a graphical interface 这是一个代码卡片生成器，可根据你上传的源码文件自动生成带有语法高亮的代码卡片，有明亮和黑暗两种主题可选择，带有图形化界面
